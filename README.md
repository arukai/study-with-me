

**study-with-me** — бұл студенттер мен өз бетінше білім алушыларға арналған өнімділікті арттыру платформасы. Жобаның басты мақсаты — пайдаланушыларға бірлесіп оқуға мүмкіндік беру, тапсырмаларды жүйелеу және оқу процесін қызықтырақ ету.

* **Проблема:** Жеке оқығанда мотивацияның төмендеуі және уақытты дұрыс басқара алмау.
* **Шешім:** Интерактивті құралдар арқылы фокусты сақтау.

## ✨ Функционал (Features)

* **Pomodoro таймері:** Жұмыс пен демалыс уақытын теңгеруге арналған кірістірілген құрал.
* **Канбан тақтасы:** Оқу жоспарларын "Орындалуда", "Аяқталды" деп бөліп, тапсырмаларды басқару.




Жобаны жергілікті компьютерде іске қосу үшін келесі қадамдарды орындаңыз:

```bash
# 1. Репозиторийді клондау
git clone [https://github.com/arukai/study-with-me.git](https://github.com/arukai/study-with-me.git)

# 2. Жоба қалтасына өту
cd study-with-me

# 3. Тәуелділіктерді орнату (npm немесе pnpm қолданыңыз)
npm install

# 4. Проектіні әзірлеу режимінде іске қосу
npm run dev



Bash
# Docker контейнерлерін жинау және іске қосу
